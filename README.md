# TuteDude-Python-Assignment-5
This repository contains Python programs demonstrating dictionaries and list slicing.
---
## Task 1: Student Marks Dictionary
This program:
- Creates a dictionary of student names and marks
- Takes student name as input
- Displays marks if student exists
- Shows error message if student not found
File: student_marks.py
---
## Task 2: List Slicing
This program:
- Creates a list from 1 to 10
- Extracts first five elements
- Reverses extracted elements using slicing
- Displays the results

File: list_slicing.py
